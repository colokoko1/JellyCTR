# JellyCTR
A Jellyfin client for the 3DS.
---
**Music only for the time being**

## Features:
* Support for all formats that Jellyfin is capable of transcoding
* Polished UI
* Lossless streaming
* Compatible with all 3DS models
* Seamless setup
## Limitations:
* No video support
* No lyric support
* Character rendering
* Navigation is a bit of a mess
